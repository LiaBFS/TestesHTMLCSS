1: o usuário digita a URL no navegador
2: DNS transforma o nome do site em endereço IP
3: navegador cria conexão TCP com o servidor
4: se for HTTPS acontece o handshake TLS pra garantir segurança
5: o navegador envia uma requisição HTTPS
6: o servidor processa e responde com HTML, CSS, etc
7: o navegador recebe e interpreta o HTML, aplicando o CSS, então executa o js e carrega recursos
8: o navegador renderiza a página
